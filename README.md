# IPO Multiplayer Chart Builder

## 📌 Overview
The IPO Multiplayer Chart Builder is a real-time collaborative web application that allows multiple users to create and edit IPO (Input, Process, Output) charts together.

Users can add items, move them between columns, and see changes live across all connected devices.

This project was created as part of a Unit 1/2 Information and Communication Technology (ICT) school assessment.

---

## 🌐 Features
- Real-time multiplayer collaboration
- Input, Process, Output (IPO) chart system
- Usernames for contributions
- Drag-and-drop organisation of items
- Auto-saving (local storage)
- Online hosting support

---

## 🛠️ Built With
- Node.js
- Express.js
- Socket.IO
- HTML, CSS, JavaScript
